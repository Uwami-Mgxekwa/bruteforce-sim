# Brute Force Simulator

A cybersecurity educational tool designed to demonstrate how brute-force login attacks work and how to defend against them. 

## Overview
- **Server Mode:** Runs a safe, simulated target server on your local machine with customizable rate limiting and account lockout defenses.
- **Attack Mode:** Scans the server using a provided wordlist to systematically guess passwords.
- **Automated Reporting:** Generates a clean, minimalistic HTML security report once a scan completes.

**⚠️ Disclaimer:** This tool is strictly for educational purposes and safe local testing. Never use it against systems you do not own or have explicit permission to test.
